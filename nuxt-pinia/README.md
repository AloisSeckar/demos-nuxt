## Nuxt + Pinia
This demo shows how to include [Pinia](https://pinia.vuejs.org/) in [Nuxt](https://nuxt.com/)

### How to use
1. `git checkout demos-nuxt project`
2. `cd nuxt-pinia`
3. `pnpm install`
4. `pnpm run dev` 
5. `localhost:3000`
