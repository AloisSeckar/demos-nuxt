## Nuxt Starter
This demo shows how [Nuxt Google Fonts](https://google-fonts.nuxtjs.org/) module can help incorporating (Google) fonts in [Nuxt](https://nuxt.com/)

### How to use
1. `git checkout demos-nuxt project`
2. `cd nuxt-fonts`
3. `pnpm install`
4. `pnpm dev` 
5. `localhost:3000`
