## Nuxt Layouts
This demo shows the concept of [layouts](https://nuxt.com/docs/4.x/directory-structure/app/layouts) in [Nuxt](https://nuxt.com/)

### How to use
1. `git checkout demos-nuxt project`
2. `cd nuxt-layouts`
3. `pnpm install`
4. `pnpm dev` 
5. `localhost:3000`
