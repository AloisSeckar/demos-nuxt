## Nuxt + Formkit
This demo shows how to include forms built and managed by [Formkit](https://formkit.com/) in [Nuxt](https://nuxt.com/)

### How to use
1. `git checkout demos-nuxt project`
2. `cd nuxt-formkit`
3. `pnpm install`
4. `pnpm dev` 
5. `localhost:3000`
