## Nuxt + Bootstrap
This demo shows how to include [Bootstrap](https://getbootstrap.com/) in [Nuxt](https://nuxt.com/)

### How to use
1. `git checkout demos-nuxt project`
2. `cd nuxt-bootstrap`
3. `npm install`
4. `npm run dev` 
5. `localhost:3000` 

You can use all the Bootstrap css classes and JS features.
