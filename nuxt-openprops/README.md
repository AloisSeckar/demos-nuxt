## Nuxt + Open Props
This demo shows [Open Props](https://open-props.style/) CSS styles in [Nuxt](https://nuxt.com/)

### How to use
1. `git checkout demos-nuxt project`
2. `cd nuxt-openprops`
3. `pnpm install`
4. `pnpm dev` 
5. `localhost:3000`
