## Nuxt + Tailwind CSS
This demo shows how to include [Tailwind](https://tailwindcss.com/) in [Nuxt](https://nuxt.com/)

### How to use
1. `git checkout demos-nuxt project`
2. `cd nuxt-tailwind`
3. `npm install`
4. `npm run dev` 
5. `localhost:3000`
