## Nuxt Starter
This demo shows using [nuxt-charts](https://nuxtcharts.com/) in [Nuxt](https://nuxt.com/)

### About
`nuxt-charts` is a module that allows including nice charts into your apps

### How to use
1. `git checkout demos-nuxt project`
2. `cd nuxt-charts`
3. `pnpm install`
4. `pnpm dev` 
5. `localhost:3000`
