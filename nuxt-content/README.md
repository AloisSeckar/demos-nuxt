## Nuxt Content
This demo shows how [Nuxt Content](https://content.nuxtjs.org/) module can help creating content pages in [Nuxt](https://nuxt.com/)

### How to use
1. `git checkout demos-nuxt project`
2. `cd nuxt-content`
3. `npm install`
4. `npm run dev` 
5. `localhost:3000`
