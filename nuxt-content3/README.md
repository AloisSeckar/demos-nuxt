## Nuxt Content v3
This demo shows how [Nuxt Content v3](https://content.nuxt.com/) module can help creating content pages in [Nuxt](https://nuxt.com/)

### How to use
1. `git checkout demos-nuxt project`
2. `cd nuxt-content3`
3. `pnpm install`
4. `pnpm dev` 
5. `localhost:3000`
