## Nuxt + DevExtreme
This demo shows how to include [DevExtreme UI library](https://js.devexpress.com/) into [Nuxt](https://nuxt.com/)

### How to use
1. `git checkout demos-nuxt project`
2. `cd nuxt-dx`
3. `pnpm install`
4. `pnpm dev` 
5. `localhost:3000` 

View in browser

### Tutorial article
https://dev.to/aloisseckar/nuxt-tutorial-7-ui-integrations-4mjm#devextreme

### Current limits/gotchas
- only work correctly with `ssr: false` setting
